> What is it? ‚ùì

AndyScript is my open-source Lua script for Stand, the Grand Theft Auto V mod menu that enhances your game experience.

For any questions or suggestions, please join my Discord server or contact me on my socials linked below.

> Features üìÉ

  • Turn off radio automatically
  • Andy's Shortcuts (create aliases for other commands, i.e. "dv" = "deletevehicle")
  • Give every weapon after joining a session
  • Heal ped to max health (only health, no armor)
  • Spawn peds inside player vehicle
  • Chaos mode
  • Andy's Spooner
  • ... and more!

> How can I try it? üí°

It's easy! You can download the latest version of AndyScript from my official GitHub post. From then, drag it into Stand's Lua Scripts folder and you're good to go! The script will automatically update, so there's no need to re-download it.
