> AndyBot 🤖

> About 🔎

AndyBot is an AI Discord bot with which you can have a conversation about anything! It uses OpenAI's GPT-3 and GPT-4 language models to generate answers. It has several dynamic settings which can be changed by any admins, such as the Model it uses, the Temperature (how creative the bot is with its answers), the maximum amount of Tokens it will "remember", and more! It also has the ability to produce images using OpenAI's Dall•E. Of course, AndyBot is capable to respond in any language, so go ahead and try it out!

> Usage 📰

To interact with AndyBot, simply tag it in your message (or if you're in DMs, simply sending a message will suffice) or use any of its build-in slash commands. Feel free to join my Discord server to interact with AndyBot!

> Commands ⌨️

The following is a list of all of AndyBot's commands. The ones marked with a 🔓 require special permissions to use them and the ones marked with a 🔒 are only available to admins.

> /help - Shows this message

> /ping - Pong! (shows the bot's latency)

> /imagine [prompt] 🔓 - Generates an image based on the prompt

> /clear-history - Clears the history of messages sent to the bot in the current server/DM

> /clear-global-history 🔒 - Clears the history of all messages sent to the bot

> /history 🔒 - Shows the history of messages sent to the bot in this server

> /global-history 🔒 - Shows the history of messages sent to the bot

> /tokens [tokens] 🔒 - Sets the amount of tokens allowed for the bot to use

> /temp [temp] 🔒 - Sets the temperature for the bot to use

> /lock-imagine [value] 🔒 - Unlocks the imagine command so anyone can use it

> /model [model] 🔒 - Sets the model for the bot to use

> /joinvc - Joins the voice channel you're in

> /play [youtube link] - Plays the audio from the YouTube video

> /leavevc - Leaves the voice channel
